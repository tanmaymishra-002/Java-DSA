## 🎯 Purpose

This repository serves as:
- A personal reference for revisiting core DSA concepts

## 🛠️ Tech Stack

- **Language:** Java
- **Approach:** Each solution focuses on correctness first, followed by time/space complexity optimization where applicable

## 🚀 Getting Started

Clone the repository and run any file directly with the Java compiler:

```bash
git clone https://github.com/tanmaymishra-002/Java-DSA.git
cd Java-DSA
javac <path-to-file>.java
java <ClassName>
```
